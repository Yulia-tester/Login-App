Login App:
A simple screen for entering email and password with a Login button and a transition to another screen. Validation check implemented. Mock data added to the code.
The following topics were studied:
1. UINavigationController
2. UIViewController Life Cycle
3. UIStackView, UITextField
4. UIImageView, UIView and form the final appearance of the interface
5. Localization
6. Button activity
